# Documentación del Módulo de Capacitación

El Centro de Capacitación es una biblioteca de videos (alojados localmente o ligados desde YouTube) acompañados de documentos adjuntos en PDF/Excel. Su propósito es servir como plataforma de entrenamiento ("Onboarding") y consulta ("Knowledge Base") para los empleados.

## Características Principales
1. **Control de Acceso (Visibility)**: Los videos no son públicos por defecto. RH puede restringir qué video se muestra a qué puesto específico o a qué usuario específico.
2. **Soporte Híbrido**: El sistema soporta subir un archivo MP4 físico al servidor, o simplemente pegar la URL de un video no listado en YouTube (lo que ahorra disco duro y ancho de banda al servidor local).
3. **Multi-Adjuntos**: Cada "curso" o "video" puede tener múltiples archivos PDF adjuntos (como manuales, infografías, presentaciones) para descargar.
