# Documentación del Módulo de Evaluación 180°

El módulo de Evaluación 180° permite a Recursos Humanos medir el desempeño del personal utilizando una metodología de retroalimentación cruzada. El término "180 grados" indica que la evaluación viaja en tres direcciones fundamentales:

1. **Evaluación de Desempeño (Hacia Abajo)**: El jefe directo evalúa las competencias técnicas y habilidades blandas de su equipo.
2. **Autoevaluación (Hacia Adentro)**: El empleado califica su propio desempeño en el semestre.
3. **Evaluación de Liderazgo (Hacia Arriba)**: El colaborador evalúa a su jefe directo exclusivamente sobre sus capacidades como líder y gestor de equipo.

Adicionalmente, el departamento de RH tiene la facultad de calificar exclusivamente sobre competencias cívicas y administrativas de los empleados.
